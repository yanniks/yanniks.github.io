How To: CyanogenMod ROM für das HTC Desire HD compilen

[caption id="" align="aligncenter" width="372"] CyanogenMod[/caption] Die alternative Android ROM CyanogenMod ist aufgrund der vielen Erweiterungen und Möglichkeiten bei den Anwendern sehr beliebt. Leider wird seit Android 4 das Desire HD nicht mehr unterstützt. Daher biete ich euch hier ein Tutorial an, wie ihr eure eigene CyanogenMod Version für das Desire HD auf Basis von Android 4.2.2 erstellen könnt. Voraussetzungen:
  • Ubuntu 12.10 64 Bit (Mac OS X funktioniert leider nicht, mit VMware Fusion könnt ihr VMs erstellen)
  • mindestens 50 GB Platz auf der Festplatte, mehr ist gut
Tutorial:
  1. Zuerst öffnet den Terminal, indem ihr auf der Dash-Startseite (Feld links oben) nach "Terminal" sucht und öffnet
  2. Gebt nun die folgenden Befehle ein: 
    sudo apt-get install openjdk-6-jdk git-core python \
  gnupg flex bison gperf build-essential \
  zip curl zlib1g-dev libc6-dev lib32ncurses5-dev ia32-libs \
  x11proto-core-dev libx11-dev lib32readline-gplv2-dev lib32z-dev \
  libgl1-mesa-dev g++-multilib mingw32 tofrodos python-markdown \
  libxml2-utils xsltproc libx11-dev:i386 lzop \
  git gnupg flex bison gperf build-essential \
  zip curl libc6-dev libncurses5-dev x11proto-core-dev \
  libx11-dev libreadline6-dev libgl1-mesa-glx \
  libgl1-mesa-dev g++-multilib mingw32 tofrodos \
  python-markdown libxml2-utils xsltproc zlib1g-dev
  3. Ladet anschließend von dieser Website das Android SDK für Linux runter: http://developer.android.com/sdk/index.html
  4. Gebt nun im Terminal den folgenden Befehl ein: 
    cd && nano .bashrc
  5. In dem anschließenden Fenster geht mit dem Pfeil nach unten bis zur untersten Stelle und fügt nun folgende Zeilen ein, danach beendet das Programm mit Steuerung + O, Enter und Steuerung + X: 
    # Android tools
export PATH=${PATH}:~/android-sdk/tools
export PATH=${PATH}:~/android-sdk/platform-tools
export PATH=${PATH}:~/bin
  6. Gebt nun nano .profile ein und macht nun das Selbe wie in Schritt 5, fügt stattdessen aber folgende Zeile ein: 
    PATH="$HOME/android-sdk-Verzeichnis/tools:$HOME/android-sdk-Verzeichnis/platform-tools:$PATH"
  7. Ladet nun die Updates in dem Fenster nach dem Eintippen von "android" herunter.
  8. Als nächstes sind mal wieder einige Kommandos im Terminal nötig: 
    mkdir ~/bin
PATH=~/bin:$PATH
curl https://dl-ssl.google.com/dl/googlesource/git-repo/repo > ~/bin/repo
chmod a+x ~/bin/repo
mkdir cm10
cd cm10/
repo init -u git://github.com/CyanogenMod/android.git -b cm-10.1
mkdir .repo/local_manifests
curl -s -o .repo/local_manifests/ace_manifest.xml https://raw.github.com/yanniks/android/cm-10.1/ace_manifest.xml
  9. Beantwortet nun die Fragen, danach könnt ihr schon fast mit dem Erstellen der ROM beginnen: 
    repo sync
cd vendor/cm/
./get-prebuilts
cd ../../
. build/envsetup.sh
lunch cm_ace-userdebug
mka bacon
  10. In dem Ordner out/target/product/ace findet ihr anschließend die fertige Build, die ihr auf euer Gerät kopieren und anschließend nutzen könnt.
Build aktualisieren: Wenn ihr eure Build aktualisieren wollt, gebt einfach hintereinander folgende Kommandos ein, danach findet ihr wieder im oben genannten Ordner eine neue Build: 
cd cm10/
rm out/target/product/ace/system/build.prop
rm out/target/product/ace/cm-*
repo sync
cd vendor/cm/
./get-prebuilts
cd ../../
. build/envsetup.sh
lunch cm_ace-userdebug
mka bacon
