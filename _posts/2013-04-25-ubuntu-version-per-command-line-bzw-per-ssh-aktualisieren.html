Nicht alle Ubuntu-Nutzer nutzen ihr System mit der Absicht, es als vollwertigen Computer zu benutzen, wie z.B. ich.

Mein Media-PC läuft auf der Basis von Ubuntu, die einzig genutzte Oberfläche ist allerdings die von XBMC, Unity habe ich deinstalliert und der einzige Kontakt zum System findet per SSH statt. Wie funktioniert nun die Aktualisierung auf Ubuntu 13.04 "Raring Ringtail"?

Das Ganze ist ganz einfach, es muss nur folgendes Kommando auf dem Server ausgeführt werden:

sudo do-release-upgrade

Den Schritten im System folgen und ihr habt die Distribution aktualisiert!
