Aktualisiert für Linux 3.9.4!

Der Linux-Kernel, der in Ubuntu 13.04 von Haus aus integriert ist, ist schon etwas älter und  hört auf den Namen Linux 3.8, welcher nicht mehr "von Haus aus" unterstützt wird. Hier zeige ich euch, wie ihr eure Ubuntu-Installation auf Version 3.9.4 aktualisieren könnt.

Dieses ist recht einfach, in einem Terminal-Fenster müsst ihr lediglich ein paar Kommandos eingeben. Ein Script, welches von UpUbuntu stammt und ich nur für die aktuelle Version aktualisert habe, übernimmt den Großteil der Arbeit.

cd /tmp

wget https://dl.dropbox.com/s/pavvgpw0631ljta/kernel-3.9.4 -O kernel-3.9.4

sudo chmod +x kernel-3.9.4

sudo sh kernel-3.9.4

sudo reboot

Anschließend ist die Arbeit getan und der Linux Kernel ist aktualisiert. Ich übernehme allerdings keine Garantie, ob das Update funktioniert. Sollte Ubuntu danach nicht mehr booten ist das nicht mein, sondern euer Problem! Erstellt deshalb sicherheitshalber ein Backup von eurer Ubuntu-Installation.
